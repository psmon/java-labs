# JAVA-LABS

Spring Boot을 통해 JAVA를 연구하고 , 자바진영의 다양한 오픈스택들을 실험하고 학습하는 프로젝트입니다.

활용되는 OpenStack들은 모두 도커로 구동하며, 샘플코드라 할지라도 로컬에서 목업이 아닌 로컬인프라를 통한 
테스트가능한 코드작성을 지원합니다.
